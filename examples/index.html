Examples

The css classes in the tachyons framework can be composed in so many different ways to build reusable components. This page will document some of the numerous examples of how to combine the classes in tachyons to build fully responsive components.

Pull requests are welcome if you would like to contribute a component or example of how to use tachyons.

A tachyon /ˈtæki.ɒn/ or tachyonic particle is a hypothetical particle that always moves faster than light.
The word comes from the Greek: ταχύς or tachys, meaning "swift, quick, fast, rapid"
