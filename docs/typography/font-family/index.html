There are several pre-defined classes for setting the typeface of a page or element. It is suggested that you customize or extend this module to suit your own needs. As tachyons is a toolkit for designing performant webpages it comes with a variety of font stacks that utilize system fonts with appropriate fallbacks.

The classes serif and sans-serif default to garamond and avenir next respectively. These can be mapped to whatever your preference is.

MDN - Font Family
src/_font-families.css 

/*

   FONT FAMILY GROUPS

*/

.sans-serif {
  font-family: 'avenir next', avenir,
                helvetica, 'helvetica neue',
                arial,
                sans-serif;
}

.serif {
  font-family: times, TimesNewRoman, "Times New Roman",
               georgia,
               serif;
}

/* From http://cssfontstack.com */

code, .code {
  font-family: Consolas,
               monaco,
               monospace;
}

.helvetica {
  font-family:  helvetica, 'helvetica neue',
                arial,
                sans-serif;
}

.bodoni {
  font-family: "Bodoni MT",
                Didot,
               "Didot LT STD",
               "Hoefler Text",
                Garamond,
                Times,
               "Times New Roman",
                serif;
}

.calisto {
  font-family: "Calisto MT",
               "Bookman Old Style", Bookman,
               "Goudy Old Style",
                Garamond,
               "Hoefler Text",
               "Bitstream Charter",
                Georgia,
                serif;
}

.garamond {
  font-family: Garamond,
               Baskerville, "Baskerville Old Face",
              "Hoefler Text",
              "Times New Roman",
               serif;
}

.times {
  font-family: Times, TimesNewRoman, "Times New Roman",
               Baskerville,
               Georgia,
               serif;
}
