# Lists

The `list` class serves as a reset for `list-style-type` to remove the default bullets.
It can be combined with `pl0` to remove the default padding on lists.

## Examples

### Ul

```.html
<ul class="list pl0">
  <li>Apples
  <li>Oranges
  <li>Banana
  <li>Golf balls
</ul>
```

### Ol

```.html
<ol class="list pl0">
  <li>One
  <li>Two
  <li>Three
  <li>Four
</ol>
```
